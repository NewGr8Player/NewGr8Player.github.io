环境安装配置

安装NodeJs

sudo apt install nodejs-legacy
sudo apt install npm

升级npm为最新版本

sudo npm install npm@latest -g

此时通过npm -v可以发现npm版本号为最新版本xxx

安装用于安装nodejs的模块n

sudo npm install -g n

安装版本

#安装官方最新版本
sudo n latest
#安装官方稳定版本
sudo n stable
#安装官方最新LTS版本
sudo n lts

更换国内npm镜像

更换后npm将变成cnpm

安装webpack

sudo npm install webpack -g

安装vue脚手架

cnpm/npm install vue-cli -g

初始化项目

vue init webpack [项目名称]

运行项目

#安装项目依赖 （添加依赖或者删除时，须重新安装）
npm install
#运行开发环境
npm run dev
#运行运营环境
npm run build

运行成功可能调用浏览器或者在浏览器中输入http://localhost:8080访问项目。
